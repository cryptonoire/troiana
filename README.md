# Troiana

Marketing site for **Troiana** — a web development agency.

🌐 **Live:** [web.troiana.net](https://web.troiana.net)

## About

A fast, zero-build single-page landing site (HTML/CSS/JS, no dependencies). Black theme, custom-built for performance.

## Run locally

Just open `index.html` in a browser — there is no build step.

```bash
# optional: serve it
python -m http.server 8000
# then visit http://localhost:8000
```

## Deploy

Static — host anywhere (Vercel, Netlify, GitHub Pages, Cloudflare Pages). Point the `web.troiana.net` subdomain at the host.
